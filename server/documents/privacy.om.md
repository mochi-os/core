# Iccitii Mochi

Beeksisni iccitii kun akkaataa **{{operator.name}}** (hojjetaan sarvera Mochi kanaa) yeroo ati akkaawuntii uumtu yookaan tajaajila kana fayyadamtu daataa dhuunfaa kee itti walitti qabu, itti fayyadamuu fi itti qooduu ibsa. {{operator.name}} too'attuu daataati.

## Maal walitti qabna

Yeroo ati galmooftee sarvera kana fayyadamtu nuti walitti qabna:

- **Daataa akkaawuntii**: maqaa fayyadamaa, mala mirkaneessaa kan filattu (teessoo email, passkeys, app mirkaneessituu, koodii deebisuu, yookaan akkaawuntii kan dhaabbata seenaa qaama sadaffaa), fi odeeffannoo profaayilaa kan ati dabaltu kamiyyuu (maqaa agarsiisuu, avatar, seenaa gabaabaa).
- **Qabiyyee ati uumtu**: maxxansa, ergaa, yaada, faayilii, fi daataa kan biraa kan ati app sarvera kana irra jiru fayyadamuun maxxansuu filattu hunda.
- **Daataa teeknikaa**: teessoo IP, odeeffannoo browser/meeshaa, fi yeroo gaaffii ati gootuu hunda. Kun tajaajila nageenyaan ittisuu fi hojjechiisuu fi miidhaa adda baasuu keessatti fayyadamna.
- **Cookies**: cookie tursiisaa yeroo seentu qabamti. Eenyummeessituu tursiisaa qoricha hin qabne qofa kuusa; cookies beeksisaaf yookaan hordoffii saayitii gidduu hin fayyadamnu.

Daataa dhuunfaa kee hin gurgurru, beeksistootatti immoo hin qoodnu.

## Mochi akkamitti daataa kee qabata (kun barbaachisaa dha)

Mochi **netiwoorkii peer-to-peer (P2P)**, malee veebsaayitii sarvera tokkoo miti. Kun akkaataa daataan kee itti yaa'u, kan iccitii keetiif barbaachisaa ta'e jijjiira:

- Qabiyyeen ati hordoftootatti, abbootii ramaddii, yookaan iddoo waliin qoodamutti maxxansitu **sarverota Mochi biroo**, kan hordoftoota ykn abbootii ramaddii sana qabachiisanitti dabarsama. {{operator.name}} sarverota inni hin hojjechiifne irraa daawwannaa qabiyyee kee balleessuu hin danda'u. Yoo maxxansa balleessite, sarverri kun gosa balleessuu peer kanneen duraan maxxansa erge sanatti erga; balleessuun kun yaaliidhaan kan raawwatamu yoo ta'u, qooda walta'iinsa peer kanaa irratti hundaa'a.
- **Eenyummaan kee (maqaa fayyadamaa, profaayilii, public keys)** yoo profaayilii uummataa filatte, daayirektoorii Mochi keessatti maxxansama, kanaaf fayyadamtoonni Mochi biroo si argachuu fi si quunnamuu danda'u.
- Qabiyyeen kallattii fi dhuunfaa (drafts, qabannoo dhuunfaa, ergaa dhuunfaa fudhattoota addaa gidduutti) sarvera kana fi sarverota fudhattoota qofa irra hafa; kun kutaa netiwoorkii P2P uummataa miti.

Gabaabumatti: wanti ati Mochi irratti namoota biroo waliin qooduu filattu kamiyyuu, akkaataa karoorsiteen, to'annaa {{operator.name}} alaatti deema. Qabiyyee ati maxxansitu akka email yookaan quunnamtii federated kamiyyuu biraa godhii ilaali.

## Qaamonni sadaffaa

Daataa dhuunfaa hir'ina ta'e qofa, yeroo ati amaloota walqabatan fayyadamtu qaamota sadaffaa kanneen waliin qoodna:

- **Tajaajila email**: email kan alaa baasamuu (koodii galmaa'uu, beeksisa, password reset) tajaajilaa SMTP kan {{operator.name}} qopheesseen ergama.
- **Tajaajiloota seenaa qaama sadaffaa**: yoo Google, GitHub, Microsoft, Facebook, yookaan tajaajilaa OAuth biroon seente, tajaajilaan sun akka ati sarvera kana seente beeka, fi teessoo email kee fi daataa profaayilii bu'uura nuuf qooda.
- **Tajaajiloota kaffaltii**: yoo bittaa app gabaa irraan raawwatte, kaffaltiin Stripe (yookaan tajaajilaan kaffaltii kan {{operator.name}} qopheesse) irraan adeemsifama. Odeeffannoo kaardii guutuu hin kuusinu.
