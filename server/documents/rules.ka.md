# Mochi სერვერის წესები

ეს წესები ვრცელდება ყველაზე, ვინც იყენებს ამ Mochi სერვერს, რომელსაც ოპერირებს **{{operator.name}}**, მიუხედავად იმისა, რომელი დომენი ან URL გამოიყენება მასზე წვდომისთვის. ისინი ვრცელდება ყველა შინაარსზე, რომელსაც აქვეყნებთ, და ყველა ურთიერთქმედებაზე ამ სერვერზე სხვა მომხმარებლებთან, მათ შორის შინაარსზე, რომელიც რეპლიცირდება ამ სერვერიდან Mochi ქსელის სხვა პირებზე.

ამ სერვერის გამოყენებით თქვენ თანხმდებით ამ წესების დაცვაზე. დარღვევებმა შეიძლება გამოიწვიოს შინაარსის წაშლა, ანგარიშის შეჩერება ან ანგარიშის დახურვა **წესებისა და პირობების** მიხედვით.

## იყავით პატივისცემითი

- არ არის დაშვებული შევიწროება, ბულინგი, მუქარა ან რომელიმე ინდივიდის ან ჯგუფის მუდმივი მიზანში მოქცევა.
- სიძულვილის ენა არ არის დაშვებული: შინაარსი, რომელიც თავს ესხმის ან ადეჰუმანიზებს ადამიანებს რასის, ეთნიკური წარმოშობის, ეროვნული წარმომავლობის, რელიგიის, კასტის, სქესის, გენდერული იდენტობის, სექსუალური ორიენტაციის, შეზღუდული შესაძლებლობების ან მძიმე ავადმყოფობის საფუძველზე, არ არის დაშვებული.
- ადამიანების ან ქონების წინააღმდეგ ძალადობისკენ წახალისება არ არის დაშვებული.
- უთანხმოება მისაღებია; შეურაცხყოფა არა. გააკრიტიკეთ იდეები და არა მათი მფლობელის ღირსება.

## დარჩით კანონის ფარგლებში

- არ არის დაშვებული შინაარსი, რომელიც უკანონოა **{{operator.jurisdiction}}**-ის კანონის მიხედვით.
- არ არის დაშვებული ბავშვთა სექსუალური ძალადობის შინაარსი. გამონაკლისები არ არსებობს და ანგარიშები, რომლებიც აქვეყნებენ ან ცდილობენ ასეთი მასალის მოპოვებას, ეცნობება შესაბამის ხელისუფლების ორგანოებს.
- არ არის დაშვებული შინაარსი, რომელიც წახალისებს ან ხელს უწყობს ტერორიზმს, ორგანიზებულ კრიმინალურ ძალადობას ან ადამიანებით ვაჭრობას.
- არ არის დაშვებული შინაარსი, რომელიც განადიდებს ან წახალისებს ცხოველებზე სისასტიკეს. რეპორტიორობა და ჟურნალისტიკა, რომელიც ასეთ სისასტიკეს გასცეს, რათა დაგმოს, მისაღებია.
- არ არის დაშვებული შინაარსი, რომელიც არღვევს სხვების საავტორო უფლებებს, სავაჭრო ნიშნებს ან სხვა ინტელექტუალური საკუთრების უფლებებს. თუ მიიღებთ შინაარსის წაშლის მოთხოვნას და მიგაჩნიათ, რომ ის შეცდომაა, დაგვიკავშირდით.

## არ მოახდინოთ სპამი ან მოტყუება

- არ არის დაშვებული მასობრივი არასასურველი კომერციული შეტყობინებები, თვალყურის დევნების/კომენტარების სპამი ან კოორდინირებული არაავთენტური ქცევა.
- არ არის დაშვებული სხვა ადამიანების, ორგანიზაციების ან როლების ისე იმიტირება, რომ შეცდომაში შეიყვანოთ. პაროდია და ნათლად მონიშნული ფანების ანგარიშები მისაღებია.
- არ არის დაშვებული ფიშინგი, მავნე პროგრამის გავრცელება ან სარწმუნოებების მოპოვების ბმულები.
- არ არის დაშვებული ხმების, რეიტინგების ან ჩართულობის მეტრიკის ხელოვნური მანიპულირება.

## პატივი ეცით კონფიდენციალურობას

- არ გამოაქვეყნოთ სხვა ადამიანების კერძო პერსონალური მონაცემები (სახლის მისამართი, ტელეფონის ნომერი, სამთავრობო იდენტიფიკატორები, კერძო შეტყობინებები) მათი თანხმობის გარეშე.
- არ გამოაქვეყნოთ რეალური ადამიანების შიშველი ან პორნოგრაფიული შინაარსი მათი თანხმობის გარეშე. ეს თანაბრად ვრცელდება როგორც ორიგინალურ ფოტოებზე ან ვიდეოზე, ისე მანიპულირებულ ან ხელოვნური ინტელექტის მიერ შექმნილ სურათებზე, რომლებზეც გამოსახულია იდენტიფიცირებადი რეალური ადამიანები.
- არ ჩაიწეროთ ან არ გააზიაროთ კერძო ერთი-ერთთან საუბრები მეორე მხარის თანხმობის გარეშე, სადაც თანხმობა მოითხოვება კანონით.
