# Katangtosan sareng sarat Mochi

Katangtosan ieu ngatur pamakean Anjeun kana layanan Mochi ieu, anu diopérasikeun ku **{{operator.name}}**. Katangtosan ieu lumaku kumaha waé Anjeun ngahontal server, kalebet ngalangkungan domain atanapi URL naon waé anu nunjuk ka dinya. Ku ngadamel akun atanapi nganggo layanan, Anjeun satuju kana katangtosan ieu. Upami Anjeun henteu satuju, tong nganggo layanan ieu.

## Layananna

Server ieu ngajalankeun Mochi, hiji platform sosial peer-to-peer. {{operator.name}} nyayogikeun infrastruktur server, manajemen akun, sareng aplikasi anu dipasang di dinya. Jaringan Mochi anu langkung lega diopérasikeun ku operator server anu mandiri di sakuliah dunya; kami ngan tanggel jawab kana server ieu wungkul.

## Akun Anjeun

- Anjeun tanggel jawab pikeun ngajaga padika autentikasi Anjeun (passkey, kodeu pamulihan, rahasia aplikasi authenticator, kredénsial OAuth) sing aman. Tong dibagikeun.
- Anjeun kedah masihan informasi anu akurat nalika ngadaptar, sareng ngajagana tetep anyar.
- Anjeun kedah cekap umur pikeun ngalebetan kontrak dumasar kana hukum **{{operator.jurisdiction}}**. Upami Anjeun di handapeun umur idin digital di nagara Anjeun, Anjeun teu meunang nganggo layanan ieu.
- Hiji jalmi tiasa nyepeng langkung ti hiji akun, tapi Anjeun teu meunang nganggo akun anu seueur pikeun nyingkahan panundaan, ngamanipulasi sora atanapi pangkat, atanapi nyamar pamaké sanés.

## Pamakean anu tiasa ditampi

Pamakean server ieu tunduk kana **Aturan server**. Naon waé anu nglanggar aturan - kalebet eusi ilegal, haharasmen, spam, panyamaran, sareng panyalahgunaan pamaké sanés - dilarang.

Anjeun nahan kapamilikan kana eusi anu Anjeun pos. Ku nerbitkeun eusi ngalangkungan server ieu, Anjeun masihan {{operator.name}} sareng jaringan Mochi anu langkung lega lisénsi anu henteu eksklusif, sadunya, bébas royalti pikeun ngahost, nyimpen, ngaréplikasi, ngirim, mintonkeun, sareng nyimpen sementara eusi éta **sakumaha anu diperyogikeun pikeun ngoperasikeun layanan** - kalebet réplikasi P2P anu didadarkeun dina wawaran **Privasi** kami. Lisénsi ieu réngsé nalika Anjeun ngahapus eusi, sanaos salinan anu parantos diréplikasi ka peer bakal tetep aya di luar kendali kami dugi ka peer-peer éta ngolah panghapusan.

## Henteu aya jaminan

Layanan disayogikeun **"sakumaha ayana"** sareng **"sakumaha sayogi"**, tanpa jaminan naon waé, jelas atanapi tersirat. Kami henteu ngajamin yén layanan moal diganggu, bébas kasalahan, aman, atanapi yén éta bakal nyumponan sarat khusus Anjeun. Pangiriman P2P sipatna usaha-pangsaéna: kami teu tiasa ngajamin yén pesen anu tangtu dugi ka peer anu tangtu dina waktos anu tangtu.
