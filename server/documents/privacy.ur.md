# Mochi پرائیویسی

یہ پرائیویسی نوٹس بیان کرتا ہے کہ **{{operator.name}}** (اِس Mochi سرور کا آپریٹر) آپ کے ذاتی ڈیٹا کو کیسے جمع، استعمال اور شیئر کرتا ہے جب آپ اکاؤنٹ بناتے ہیں یا سروس کا استعمال کرتے ہیں۔ {{operator.name}} ڈیٹا controller ہے۔

## ہم کیا جمع کرتے ہیں

جب آپ سائن اپ کرتے ہیں اور اِس سرور کا استعمال کرتے ہیں تو ہم جمع کرتے ہیں:

- **اکاؤنٹ ڈیٹا**: ایک صارف نام، آپ کے منتخب کردہ authentication طریقے (ای میل ایڈریس، passkeys، authenticator ایپ، recovery کوڈز، یا کسی فریقِ ثالث لاگ ان فراہم کنندہ کا اکاؤنٹ)، اور آپ کی شامل کردہ کوئی بھی پروفائل معلومات (display نام، avatar، biography)۔
- **آپ کا تخلیق کردہ مواد**: پوسٹس، پیغامات، تبصرے، فائلیں، اور کوئی بھی دیگر ڈیٹا جسے آپ اِس سرور پر انسٹال شدہ ایپس کے ذریعے شائع کرنا منتخب کرتے ہیں۔
- **تکنیکی ڈیٹا**: IP ایڈریس، براؤزر/ڈیوائس کی معلومات، اور آپ کی درخواستوں کے timestamps۔ یہ سروس کو محفوظ اور فعال رکھنے، اور بدسلوکی کا پتہ لگانے کے لیے استعمال ہوتا ہے۔
- **Cookies**: جب آپ سائن ان کرتے ہیں تو ایک session cookie سیٹ کی جاتی ہے۔ یہ صرف ایک opaque session شناخت کنندہ ذخیرہ کرتی ہے؛ ہم cookies کا استعمال اشتہارات یا cross-site ٹریکنگ کے لیے نہیں کرتے۔

ہم آپ کا ذاتی ڈیٹا فروخت نہیں کرتے، اور اِسے اشتہار دینے والوں کے ساتھ شیئر نہیں کرتے۔

## Mochi آپ کا ڈیٹا کیسے سنبھالتا ہے (یہ اہم ہے)

Mochi ایک **peer-to-peer (P2P) نیٹ ورک** ہے، نہ کہ سنگل سرور پلیٹ فارم۔ یہ آپ کے ڈیٹا کے بہاؤ کو اِس طرح بدلتا ہے کہ یہ آپ کی پرائیویسی کے لیے اہمیت رکھتا ہے:

- وہ مواد جو آپ followers، subscribers یا shared spaces میں شائع کرتے ہیں، وہ **دیگر Mochi سرورز پر replicate کیا جاتا ہے** جو اُن followers یا subscribers کو host کرتے ہیں۔ {{operator.name}} اپنے سرور کے علاوہ کسی سرور سے آپ کے مواد کی کاپیاں حذف نہیں کر سکتا۔ اگر آپ کوئی پوسٹ حذف کرتے ہیں تو یہ سرور اُن peers کو حذف کرنے کا نوٹس بھیجے گا جنہیں اِس نے پہلے پوسٹ بھیجی تھی، لیکن یہ حذف کرنا best-effort ہے اور اُن peers کے تعاون پر منحصر ہے۔
- اگر آپ public پروفائل منتخب کرتے ہیں تو آپ کی **شناخت (صارف نام، پروفائل، public keys)** Mochi ڈائریکٹری میں شائع کی جاتی ہے، تاکہ دیگر Mochi صارفین آپ کو تلاش اور آپ سے رابطہ کر سکیں۔
- براہ راست، نجی مواد (ڈرافٹس، نجی نوٹس، مخصوص وصول کنندگان کے درمیان نجی پیغامات) صرف اِس سرور اور وصول کنندگان کے سرورز پر رہتا ہے؛ یہ public P2P نیٹ ورک کا حصہ نہیں ہے۔

مختصر یہ کہ: جو کچھ بھی آپ Mochi پر دوسرے لوگوں کے ساتھ شیئر کرنے کا انتخاب کرتے ہیں، وہ ڈیزائن کے اعتبار سے {{operator.name}} کے قابو سے باہر سفر کرے گا۔ شائع کیے گئے مواد کو ای میل یا کسی بھی دیگر federated مواصلات کی طرح سمجھیں۔

- **ای میل سروس**: outbound ای میلز (سائن اپ کوڈز، notifications، پاس ورڈ resets) {{operator.name}} کے configure کردہ SMTP فراہم کنندہ کے ذریعے بھیجی جاتی ہیں۔
- **فریقِ ثالث لاگ ان فراہم کنندگان**: اگر آپ Google، GitHub، Microsoft، Facebook یا کسی اور OAuth فراہم کنندہ سے سائن ان کرتے ہیں تو وہ فراہم کنندہ جانتا ہے کہ آپ نے اِس سرور پر سائن ان کیا، اور آپ کا ای میل ایڈریس اور بنیادی پروفائل ڈیٹا ہمارے ساتھ شیئر کرتا ہے۔
