## Naon anu kami kumpulkeun

Nalika Anjeun ngadaptar sareng nganggo server ieu, kami ngumpulkeun:

- **Data akun**: ngaran pamaké, padika autentikasi anu Anjeun pilih (alamat email, passkey, aplikasi authenticator, kodeu pamulihan, atanapi akun ti panyadia login pihak katilu), sareng informasi profil naon waé anu Anjeun tambihan (ngaran tampilan, avatar, biografi).
- **Eusi anu Anjeun damel**: pos, pesen, koméntar, koropak, sareng data sanés anu Anjeun pilih pikeun dipedalkeun ngalangkungan aplikasi anu dipasang dina server ieu.
- **Data téknis**: alamat IP, informasi panyungsi/parangkat, sareng cap waktu pikeun pamundut anu Anjeun damel. Ieu dianggo pikeun ngajaga kaamanan sareng oprasi layanan, sareng pikeun ngadeteksi panyalahgunaan.
- **Cookie**: cookie sési disetél nalika Anjeun lebet. Cookie éta ngan ukur nyimpen pananda sési anu opaque; kami henteu nganggo cookie pikeun iklan atanapi nyukcruk lintas-situs.

Kami henteu ngajual data pribados Anjeun, sareng kami henteu ngabagikeunna ka pangiklan.

## Kumaha Mochi nanganan data Anjeun (ieu penting)

Mochi mangrupi **jaringan peer-to-peer (P2P)**, sanés platform server tunggal. Hal éta ngarobih cara data Anjeun ngalir ku cara anu penting pikeun privasi Anjeun:

- Eusi anu Anjeun publikasikeun ka anu nuturkeun, palanggan, atanapi rohangan anu dibagi **diréplikasi ka server Mochi sanés** anu nampung anu nuturkeun atanapi palanggan éta. {{operator.name}} teu tiasa ngahapus salinan eusi Anjeun ti server anu sanés diopérasikeun ku anjeunna. Upami Anjeun ngahapus hiji pos, server ieu bakal ngintunkeun bewara panghapusan ka peer anu sateuacanna parantos nampi pos éta, tapi panghapusan éta sipatna usaha-pangsaéna sareng gumantung kana karjasama peer-peer éta.
- **Identitas Anjeun (ngaran pamaké, profil, konci umum)** dipedalkeun ka diréktori Mochi upami Anjeun milih profil umum, ku kituna pamaké Mochi anu sanés tiasa milari sareng ngahubungan Anjeun.
- Eusi pribados langsung (rancangan, catetan pribados, pesen pribados antara panampi anu tangtu) ngan tetep aya dina server ieu sareng server panampina; éta sanés bagian tina jaringan P2P umum.

Pondokna: naon waé anu Anjeun pilih pikeun dibagi ka batur dina Mochi bakal ngalangkungan kendali {{operator.name}}, saluyu sareng desainna. Perlakukeun eusi anu Anjeun publikasikeun sapertos email atanapi komunikasi terfederasi sanésna.
