## ကျွန်ုပ်တို့ စုဆောင်းသည့်အရာ

သင် ဤဆာဗာတွင် အကောင့်ဖွင့်ပြီး အသုံးပြုသည့်အခါ ကျွန်ုပ်တို့ စုဆောင်းသည်မှာ-

- **အကောင့်အချက်အလက်**- အသုံးပြုသူအမည်၊ သင်ရွေးချယ်သော အထောက်အထားစစ်ဆေးနည်းများ (အီးမေးလ်လိပ်စာ၊ passkey များ၊ authenticator အက်ပ်၊ ပြန်လည်ရယူရန်ကုဒ်များ၊ သို့မဟုတ် ပြင်ပ login ပံ့ပိုးသူထံမှ အကောင့်)၊ နှင့် သင်ထည့်သွင်းသည့် ပရိုဖိုင်အချက်အလက် (ဖော်ပြသည့်အမည်၊ ပရိုဖိုင်ပုံ၊ ကိုယ်ရေးအကျဉ်း) တို့ဖြစ်သည်။
- **သင်ဖန်တီးသည့် အကြောင်းအရာများ**- ပို့စ်များ၊ မက်ဆေ့ဂျ်များ၊ မှတ်ချက်များ၊ ဖိုင်များ၊ နှင့် ဤဆာဗာတွင် တပ်ဆင်ထားသော အက်ပ်များမှတစ်ဆင့် သင်ထုတ်ဝေရန်ရွေးချယ်သည့် အခြားအချက်အလက်များ။
- **နည်းပညာဆိုင်ရာ အချက်အလက်**- IP လိပ်စာ၊ browser/စက်ပစ္စည်းအချက်အလက်၊ နှင့် သင်ပြုလုပ်သော တောင်းဆိုမှုများ၏ အချိန်တံဆိပ်များ။ ဤအချက်အလက်များကို ဝန်ဆောင်မှုကို လုံခြုံစွာ ဆက်လက်လည်ပတ်စေရန် နှင့် မလျော်ကန်သော အသုံးပြုမှုကို ရှာဖွေဖော်ထုတ်ရန် အသုံးပြုသည်။
- **Cookies**- သင် login ဝင်သည့်အခါ session cookie တစ်ခု သတ်မှတ်ပေးထားသည်။ ၎င်းတွင် မြင်သာသော session ID တစ်ခုသာ သိမ်းထားသည်။ ကြော်ငြာများ သို့မဟုတ် cross-site ခြေရာခံခြင်းအတွက် cookies များ မသုံးပါ။

ကျွန်ုပ်တို့သည် သင်၏ ကိုယ်ရေးကိုယ်တာအချက်အလက်ကို မရောင်းချပါ၊ ကြော်ငြာရှင်များနှင့်လည်း မမျှဝေပါ။

## Mochi သည် သင့်အချက်အလက်ကို မည်သို့ ကိုင်တွယ်သည် (ဤအချက်သည် အရေးကြီးသည်)

Mochi သည် ဆာဗာတစ်ခုတည်းဖြင့် လည်ပတ်သော platform မဟုတ်ဘဲ **peer-to-peer (P2P) ကွန်ရက်** ဖြစ်သည်။ ၎င်းသည် သင်၏ ကိုယ်ရေးကိုယ်တာအချက်အလက် စီးဆင်းပုံကို အဓိကကျသော နည်းလမ်းများဖြင့် ပြောင်းလဲစေသည်-

- သင် followers များ၊ subscribers များ၊ သို့မဟုတ် မျှဝေထားသော နေရာများသို့ ထုတ်ဝေသော အကြောင်းအရာများသည် ထို followers များ သို့မဟုတ် subscribers များကို လက်ခံထားသော **အခြား Mochi ဆာဗာများသို့ ကူးယူ ဖြန့်ဝေသည်**။ {{operator.name}} သည် ၎င်းမလည်ပတ်သော ဆာဗာများရှိ သင်၏ အကြောင်းအရာမိတ္တူများကို ဖျက်ပစ်နိုင်ခြင်းမရှိပါ။ သင် ပို့စ်တစ်ခုကို ဖျက်ပါက ဤဆာဗာသည် ယခင်က ပို့စ်ပို့ထားသော peer များထံ ဖျက်သိမ်းရန်အကြောင်းကြားချက်ကို ပို့မည်ဖြစ်သော်လည်း၊ ဖျက်သိမ်းခြင်းသည် အကောင်းဆုံးကြိုးပမ်းမှုသာဖြစ်ပြီး၊ ထို peer များ၏ ပူးပေါင်းဆောင်ရွက်မှုပေါ်တွင် မူတည်သည်။
- သင် အများပြည်သူ မြင်ရသော ပရိုဖိုင်ကို ရွေးချယ်ပါက သင်၏ **အထောက်အထား (အသုံးပြုသူအမည်၊ ပရိုဖိုင်၊ public key များ)** ကို Mochi directory တွင် ထုတ်ဝေသည်။ ၎င်းကြောင့် အခြား Mochi အသုံးပြုသူများသည် သင့်ကို ရှာဖွေ၍ ဆက်သွယ်နိုင်ပါသည်။
- တိုက်ရိုက်ပို့သော သီးသန့်အကြောင်းအရာများ (drafts၊ သီးသန့်မှတ်စုများ၊ သတ်မှတ်ထားသော လက်ခံသူများကြားရှိ သီးသန့်မက်ဆေ့ဂျ်များ) သည် ဤဆာဗာနှင့် လက်ခံသူတို့၏ ဆာဗာတွင်သာ တည်ရှိနေသည်။ ၎င်းတို့သည် အများပြည်သူ P2P ကွန်ရက်၏ အစိတ်အပိုင်း မဟုတ်ပါ။

အကျဉ်းချုပ်အားဖြင့်- Mochi တွင် အခြားသူများနှင့် မျှဝေရန် သင်ရွေးချယ်လိုက်သော မည်သည့်အရာမဆို အကြံအစည်အရ {{operator.name}} ၏ ထိန်းချုပ်မှုထက်ကျော်လွန်၍ သွားရောက်နိုင်သည်။ သင်ထုတ်ဝေသော အကြောင်းအရာများကို အီးမေးလ် သို့မဟုတ် အခြား federated ဆက်သွယ်ရေးနှင့် တူညီစွာ သတိရှိပါ။
