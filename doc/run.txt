To run on Linux:

 1. Ensure that the machine can send email from the command line using the "mail" command or similar.
 2. git clone git@github.com:mochi-project/core.git
 3. cd core
 4. make
 5. sudo cp --update=none install/etc/mochi/mochi.conf /etc/mochi/mochi.conf
 6. ./mochi-server
 7. In a web browser, go to http://localhost:8080/
 8. Follow the instructions in your web browser.
